# References: Concept Labeling and Metadata Standards

1. [Simple Knowledge Organization System](https://en.wikipedia.org/wiki/Simple_Knowledge_Organization_System) - Wikipedia - Describes the W3C SKOS standard for representing controlled vocabularies with prefLabel, altLabel, and broader/narrower relationships, the exact model this chapter uses to explain preferred and alternate labels.

2. [Dublin Core](https://en.wikipedia.org/wiki/Dublin_Core) - Wikipedia - Covers the 15-element Dublin Core metadata standard for describing resources like books and datasets, the standard this chapter applies to tagging a learning graph file itself as a discoverable resource.

3. [ISO/IEC 11179](https://en.wikipedia.org/wiki/ISO/IEC_11179) - Wikipedia - Explains the international standard for metadata registries and data element definitions, the source of this chapter's five-criteria test (precise, concise, distinct, non-circular, unencumbered) for a well-formed concept definition.

4. The Organization of Information (4th Edition) - Arlene G. Taylor and Daniel N. Joudrey - Libraries Unlimited - A foundational library-science textbook covering controlled vocabularies, indexing, and metadata description; grounds this chapter's naming and cataloging vocabulary in professional information-organization practice.

5. Introduction to Controlled Vocabularies: Terminology for Art, Architecture, and Other Cultural Works (Updated Edition) - Patricia Harpring - Getty Research Institute - Explains how to construct and maintain controlled vocabularies with preferred terms and synonym rings, directly informing this chapter's preferred-label/alternate-label distinction.

6. [SKOS Primer](https://www.w3.org/TR/skos-primer/) - W3C - The official W3C guide to representing thesauri, taxonomies, and controlled vocabularies in SKOS, walking through prefLabel, altLabel, broader, narrower, and related properties covered in this chapter's SKOS section.

7. [DCMI Metadata Terms](https://www.dublincore.org/specifications/dublin-core/dcmi-terms/) - Dublin Core Metadata Initiative - The authoritative specification listing all Dublin Core elements (Title, Creator, Subject, etc.), the reference this chapter draws on for its Dublin Core Metadata Explorer diagram.

8. [ANSI/NISO Z39.19 Guidelines](https://www.niso.org/publications/ansiniso-z3919-2005-r2010-guidelines-construction-format-and-management-monolingual) - NISO - The formal standard for constructing and managing monolingual controlled vocabularies, underpinning this chapter's discussion of controlled vocabularies as a governance discipline distinct from the SKOS file format.

9. [Frequently Asked Questions](https://asindexing.org/about-indexing/frequently-asked-questions/) - American Society for Indexing - Explains how professional back-of-book indexes are built and why they require conceptual understanding, illuminating this chapter's claim that a textbook index can be generated automatically from well-labeled concepts.

10. [Understanding Metadata: What Is Metadata, and What Is It For?](https://www.niso.org/publications/understanding-metadata-2017) - NISO - A free primer surveying metadata types and standardization practices, giving broader context for this chapter's specific standards (SKOS, ISO/IEC 11179, Dublin Core) within the wider metadata landscape.
