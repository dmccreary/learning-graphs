#### 1. What term names the following idea: “A method for collecting evidence about a learner’s knowledge, skill, performance, or progress.”?

<div class="upper-alpha" markdown>
1. Feedback Analysis
2. Assessment
3. Course Evaluation
4. Learning Touchpoint
</div>

??? question "Show Answer"
    The correct answer is **B**. Assessment is a method for collecting evidence about a learner’s knowledge, skill, performance, or progress. The chapter's terminology supports the same distinction in practice: After a quiz, assessment can help decide whether the learner needs review, practice, or a new challenge. Feedback Analysis, Course Evaluation, and Learning Touchpoint name neighboring ideas, but none has this defining purpose. The decisive clue is the function described in the stem, not merely the fact that all four terms belong to Assessment, Feedback, and Quizzes.

    **Concept Tested:** Assessment
