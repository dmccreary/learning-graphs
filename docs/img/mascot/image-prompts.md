# Image Prompts: Axiom the Owl

I am about to ask you to generate eight different poses for a book mascot.
Please use a consistent drawing style for all eight images.

These prompts are provided for **regeneration reference only** — the pose
images already exist in this directory (copied from the source Axiom
character in [dmccreary/intelligent-textbooks](https://github.com/dmccreary/intelligent-textbooks)).
Each prompt is fully self-contained so it can be used independently.

## Character Lock (include in every prompt)

```
A friendly academic owl mascot named Axiom. Compact rounded body. Deep
academic blue feathers with cream face and chest. Large golden-yellow eyes
framed by dark brow markings resembling scholarly glasses. Warm amber-orange
beak and talons. Small gold collar with a bright red tie. Clean vector-style
illustration. Soft shading. Flat colors. Professional educational tone.
Transparent background. Maintain identical character proportions, facial
structure, color palette, and accessory placement as previous approved
Axiom mascot renderings.
```

Negative prompt (avoid): photorealistic, 3D render, noisy, blurry,
low-resolution, harsh gradients, neon colors, complex background, scenery,
text, watermark, logo, extra characters, distorted anatomy, extra wings,
extra eyes, extra beak, deformed tie, inconsistent colors.

Recommended sizes: 1024x1024 (square poses) or 1024x1536 (tall poses).
Output format: transparent PNG.

## Pose Variants

### 1. Neutral Pose (`neutral.png`)

**Use:** General sidebars, standard instructional guidance within chapter body text and callouts.

[CHARACTER LOCK] Pose: Wings relaxed at sides. Upright balanced stance. Gentle closed-beak smile. Eyes attentive and calm. Emotion: Steady, wise, patient, supportive. Lighting: Soft, neutral lighting. Background: Transparent.

### 2. Welcome Pose (`welcome.png`)

**Use:** Chapter openings, start of book/module, major transitions.

[CHARACTER LOCK] Pose: Wings open outward in a gentle welcoming arc. Upright posture. Slight forward lean. Bright friendly eyes. Warm confident smile. Emotion: Cheerful, encouraging, welcoming, supportive. Lighting: Soft, even lighting. No dramatic shadows. Background: Transparent.

### 3. Explain Pose (`explain.png`)

**Use:** Concept clarification sections, walkthrough steps, key ideas. (Extra pose beyond the base 7 — distinguishes "explaining a concept" from the more reflective Thinking pose.)

[CHARACTER LOCK] Pose: One wing slightly raised as if explaining. Slight forward lean. Beak slightly open as if speaking. Focused eyes. Emotion: Clear, confident, instructional. Lighting: Soft academic lighting. Background: Transparent.

### 4. Thinking Pose (`thinking.png`)

**Use:** Key concepts, analysis, reflection prompts, metacognitive coaching moments.

[CHARACTER LOCK] Pose: One wing touching chin thoughtfully. Head slightly tilted. Eyebrows gently angled inward. Soft contemplative expression. Emotion: Curious, reflective, analytical, patient. Lighting: Soft directional lighting from upper left. Background: Transparent.

### 5. Tip Pose (`tip.png`)

**Use:** Pro tips, best practices, insights, MicroSim exploration guidance.

[CHARACTER LOCK] Pose: One wing raised with a single feather extended upward as if giving a helpful tip. Slight knowing smile. Eyes slightly narrowed in insight. Upright posture. Optional element: small subtle sparkle or tiny lightbulb icon near raised feather (minimal, flat). Emotion: Insightful, strategic, helpful, confident. Lighting: Soft and bright. Background: Transparent.

### 6. Warning Pose (`warning.png`)

**Use:** Common mistakes, misconceptions, pitfalls — gentle, never alarming.

[CHARACTER LOCK] Pose: Both wings slightly raised in a gentle caution gesture. Calm but concerned expression. Eyes focused. Optional element: small minimal exclamation mark icon floating near one wing (flat, not alarming). Emotion: Caring, protective, steady — not alarmist. Lighting: Neutral lighting. Background: Transparent.

### 7. Encouraging Pose (`encouraging.png`)

**Use:** Hard parts of the chapter, reassurance, persistence messaging.

[CHARACTER LOCK] Pose: Wing giving a thumbs-up gesture. Warm supportive smile. Slight lean toward viewer. Emotion: Encouraging, optimistic, reassuring. Lighting: Warm soft lighting. Background: Transparent.

### 8. Celebration Pose (`celebration.png`)

**Use:** End-of-chapter mastery, completion acknowledgements, milestones.

[CHARACTER LOCK] Pose: Wings fully spread wide. Upright confident posture. Bright joyful eyes. Big dignified smile. Optional elements: subtle gold confetti or small sparkles around wings (minimal, no excessive cartoon effects). Emotion: Proud, celebratory, affirming. Lighting: Warm golden glow accent. Background: Transparent.
