    Quiz: Concept Taxonomies and Ontologies

    Test your understanding of concept taxonomies and ontologies with these review questions.

    1. What term names the following idea: “A classification structure that groups concepts into meaningful categories for organization, navigation, and analysis.”?

    1. Concept Taxonomy
    2. Taxonomy Category
    3. Taxonomy Distribution Report
    4. Taxonomy Legend
    Show Answer

    The correct answer is A. Concept Taxonomy is a classification structure that groups concepts into meaningful categories for organization, navigation, and analysis. The chapter's terminology supports the same distinction in practice: A taxonomy review can use concept taxonomy to decide whether concepts are grouped clearly and evenly. Taxonomy Category, Taxonomy Distribution Report, and Taxonomy Legend name neighboring ideas, but none has this defining purpose. The decisive clue is the function described in the stem, not merely the fact that all four terms belong to Concept Taxonomies and Ontologies.

    1. A graph attribute that assigns nodes to a shared visual or semantic group.
    2. The assignment of each concept to one or more taxonomy categories.
    3. A formal model of a domain that defines classes, properties, and relationships among concepts or entities.
    4. A taxonomy whose categories contain enough concepts to be useful without concentrating too many concepts in one group.

    The correct answer is C. The requirement points to Ontology: A formal model of a domain that defines classes, properties, and relationships among concepts or entities. In a balanced classification scheme for a large concept set, that function is what separates it from Node Group, Concept-to-Taxonomy Mapping, and Balanced Taxonomy. Those alternatives may participate in the same workflow, but substituting one of them would change the role the question asks the practitioner to identify.
